# OnPremise SRT Server — Documentación

Web oficial: https://streamrus.com/

Este repositorio contiene **solo documentación** de **OnPremise SRT Server**.

OnPremise SRT Server es una solución on-premises (software + imagen de S.O.) orientada a flujos profesionales de contribución y distribución broadcast/IPTV, centrada en transporte **SRT** y **UDP/RTP** de **MPEG-TS**.

> Nota: orientado a despliegues en **hardware físico**.

---

## Qué hay en este repositorio

- Documentación y FAQ prácticas (EN/ES)
- Workflows típicos (SRT ↔ UDP/RTP)
- Notas de red y multicast para cabeceras
- Resumen de API REST (Swagger)
- Manuales completos en Markdown y PDFs originales
